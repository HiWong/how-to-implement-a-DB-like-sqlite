# how-to-implement-a-DB-like-sqlite


##Description
在这个项目中我们会学习SQLite源码并仿制其做成一个完整的数据库.

整个过程中我们会对SQLite进行多次demo, 不断的加入新功能, 直到其成为一个安全, 快速, 功能较为完整的数据库.
